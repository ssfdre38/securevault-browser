WHAT EACH SCRIPT DOES
═══════════════════════════════════════════════════════════════

📥 setup.sh
   └─ Installs depot_tools (Chromium build tools)
   └─ Installs system dependencies (build-essential, libraries)
   └─ Downloads Chromium source code (~20GB)
   └─ Runs gclient sync to fetch dependencies
   └─ Applies SecureVault privacy patches
   └─ Copies branding files
   ⏱️  Time: 30-60 minutes
   💾 Disk: ~20GB download

🔨 build.sh
   └─ Creates build configuration (args.gn)
   └─ Disables Google API keys
   └─ Disables telemetry and crash reporting
   └─ Enables privacy features
   └─ Generates ninja build files with gn
   └─ Compiles browser with all CPU cores
   └─ Outputs binary to src/out/SecureVault/chrome
   ⏱️  Time: 1-3 hours (first build)
   💾 Output: ~200MB binary + resources

🚀 run.sh
   └─ Launches browser from src/out/SecureVault/chrome
   └─ Uses privacy-focused user data directory
   └─ Applies privacy command-line flags:
      • --disable-background-networking
      • --disable-sync
      • --disable-translate
      • --dns-over-https-mode=secure
      • --force-webrtc-ip-handling-policy
      • And 15+ more privacy flags

🎨 generate-icons.sh
   └─ Converts SVG icons to PNG (multiple sizes)
   └─ Creates Windows ICO file (multi-size)
   └─ Creates macOS ICNS file (multi-size)
   └─ Generates sizes: 16, 24, 32, 48, 64, 128, 256, 512
   ⚠️  Requires: inkscape or imagemagick
